NUMBERING SYSTEMS


What is Number systems?

A numeral system (or system of numeration) is a writing system for expressing numbers, that is, a mathematical notation for representing numbers of a given set, using digits or other symbols in a consistent manner.





Experience:

Number systems in general were really easy, it was just addition and subtraction. We did three types of conversions:

  • - BINARY-decimal conversion
  • - OCTAL-decimal conversion
  • - HEXADECIMAL-decimal conversion

    • Binary and octal conversions were really easy as we could just change them using numbers, addition and subtractions were really easy too, but hexadecimal were a little hard to understand as we had to use letters after 9 and to was hard to do addition and subtraction with it. In general, I would want to learn more about it like division and multiplication, I would actually like to use it in real life, not just simulations. Using what you learn in real life makes the topic more real and fun as you have to use logic to solve problems.

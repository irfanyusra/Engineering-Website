TURTLE


What is Turtle?

Turtle graphics is a term in computer graphics vector graphics using a relative cursor (the "turtle") upon a Cartesian plane. Turtle graphics is a key feature of the Logo programming language.


Winter Scene




Experience:

Turtle was one of my favourite things in this course as we could be creative. Our class had to make a winter scene using turtle commands and animate some aspects of it. I made a whole scenery and animated a bunch of stuff, like I made my snowman wink, I made snowflakes fall, the snowman on the sledge went across the screen and the smoke from the cabin was animated. I learnt a lot from turtle, like using for loops and stuff. The best thing about doing this project was logic and math, math was one of the main things because a lot if angles were used.

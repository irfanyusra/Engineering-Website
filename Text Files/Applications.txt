		<h1 id="heading"> <center>Logic Friday</center></h1>
		<br>
		<h id="subH"><b>What is Logic Friday?</b></h>
		<p id="one">Logic Friday is a freeware tool for students, hobbyists, and
			engineers who work with legacy digital logic circuits based on standard IC packages.
		<br> With Logic Friday you can:
		</p>

		<li id="one" style="padding-left:25%;">	•	Enter a logic function as a truth table, an equation, or a gate diagram and convert them into other.</li>
		<li id="one" style="padding-left:25%;">	•	Simplify logic functions</li>
		<li id="one" style="padding-left:25%;">	•	Compare logic functions</li>
		<li id="one" style="padding-left:25%;">	•	Generate new functions as logical combinations of others</li>
		<br>
		<img style="padding-left:370px;" src="logicF.png" alt="Logic Friday" >

		<h id="subH"><b>Experience:</b></h>
		<p id="one">My Experience with Logic Friday was really good, it is a really
			easy program to use. The user should just know the basics, the best part
			about this program is that it converts Boolean Equation or Truth Table
			into Logic Diagram and asks for the gates to be used. This program also
			simplifies the Boolean Equation or the Logic Diagrams. I learnt how to use
			this program in 2 minutes, and I used this program for my Traffic Light
			assignment. </p>
